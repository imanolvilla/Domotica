Los cuatros pilares de la Domótica


Confort

El hecho de poder controlar las bombillas y equipos de climatización y sistemas de ventilación desde nuestros dispositivos móviles, son solo algunos ejemplos que contribuyen a mejorar nuestro confort y comodidad.

Seguridad

Aplicada a las Smart Home. En este ámbito nos encontramos desde alarmas de intrusión para evitar robos, hasta detectores de todo tipo que velan por nuestra seguridad: fuego, humo, gas, fugas y escapes de agua.

Energético

Seguridad domótica

La seguridad domótica, también denominada seguridad del hogar inteligente, es el conjunto de tecnologías y sistemas empleados para proteger una vivienda mediante la integración de dispositivos y sistemas electrónicos conectados a la red.

Sistemas domóticos inalámbricos

En este tipo de instalaciones de sistemas domóticos para viviendas no se usan cables para transmitir la señal, sino que la información se envía a través de ondas de radiofrecuencia, que se encargan de interpretar las diversas órdenes que se comparten.

Tiempo entre Cliente-Servidor

Nuestros sistemas domóticos cuenta con un tiempo de 200ms en respuesta, modelo cliente-servidor.

Calendario de Inicio

Nuestro calendario de inicio puedes programar ciertos parametros para que se realicen de manera automática.

Iluminación inteligente para el hogar

Implica la instalación de un sistema tecnológico para la gestión de las bombillas de nuestro hogar mediante un smartphone, un mando a distancia, un hub de control o incluso nuestra propia voz.

Sistemas domóticos con cable compartido

También conocido como powerline, se trata de un sistema domótico en el que se emplea el cable de alimentación eléctrica normal, para hacer el envío de la señal de control.

Sistema de Pruebas

Nuestro sistema de prueba, se implementa los sensores, actuadores, interfaz y controlador para realizar testing de instalación y funcionamiento.
